## difyplugin_yomitoku

### Description
日本語特化OCR「[YomiToku](https://github.com/kotaro-kinoshita/yomitoku)」のDifyプラグイン  
Unofficial Dify plugin for YomiToku, japanese ocr.

### 本プロジェクトの目標
- YomiToku公式のプラグインとしてDifyのマーケットプレイスに掲載されること。
